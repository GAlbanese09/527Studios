# 527 Studios — James Albanese

Graphic design portfolio & storefront for James Albanese.

**Stack:** React 19 + Vite + Cloudflare Pages

## Local Development

```bash
npm install
npm run dev
```

## Build & Deploy

```bash
npm run build
```

Deploys to Cloudflare Pages via GitHub auto-deploy. Build output is in `dist/`.

## Cloudflare Pages Settings

- **Build command:** `npm run build`
- **Build output directory:** `dist`
- **Root directory:** `/`
